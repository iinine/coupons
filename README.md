## 使用方法
按照步骤一步一步来就行。
```bash
# -- Start

git clone git@github.com:youmengme/coupons.git
yarn 
# if 你要使用开发环境实现修改代码热更新
yarn dev:weapp
# else 你只想打个包
yarn build:weapp

# 使用微信开发者工具 导入小程序 选择你clone下来的文件夹下的dist目录 

# -- End

# 如果有不懂的地方 欢迎群里 @凌晨三点
```
